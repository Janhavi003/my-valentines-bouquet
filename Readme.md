# 💐 My Valentine’s Bouquet

An interactive, aesthetic bouquet builder that lets users design a custom Valentine’s bouquet by selecting flowers, choosing a vase, adding a personal message, and exporting the result as a shareable card.

👉 **Live Demo:** https://janhavi003.github.io/my-valentines-bouquet/
👉 **Source Code:** https://github.com/Janhavi003/my-valentines-bouquet

---

## Features

- **Multiple flowers in one vase** with dynamic visual stacking  
- **Carousel-based selection** for flowers and vases  
- **Personalized message** with “To / From” fields  
- **Real-time preview** as you build your bouquet  
- **Save multiple named bouquets** using `localStorage`  
- **Reload saved bouquets** instantly  
- **Export bouquet as an image card** (no UI clutter)  
- **Confetti animation** on save for delightful feedback  
- **Toast notifications** for user actions  
- **Soft pastel Valentine theme** with smooth micro-interactions  

---

## Built With

- **HTML5** – semantic structure  
- **CSS3** – custom properties, animations, responsive layout  
- **Vanilla JavaScript (ES Modules)** – no frameworks  
- **html2canvas** – export bouquet as an image  
- **canvas-confetti** – celebratory UI feedback  


---

## Key Technical Highlights

- Modular JavaScript architecture using ES modules  
- Centralized state management for bouquet data  
- Persistent storage with `localStorage`  
- Dynamic DOM rendering and layered visual composition  
- Export-only rendering area for clean image generation  
- UX-focused micro-interactions and animations
